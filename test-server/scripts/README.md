# Test Server Scripts

**🤖 AGENT INSTRUCTIONS**

This directory contains utility scripts and templates for the `test-server` environment.

## Contents

- `standardize-prompts.js`: A script used to standardize formatting, update tool lists, and ensure consistency across the hundreds of markdown test prompts in the `test-server` directories.
- `prompt-template.md`: The base markdown template used when generating new testing prompts.

When asked to update testing structures or create new tests, refer to these files to maintain consistency across the test suite.
