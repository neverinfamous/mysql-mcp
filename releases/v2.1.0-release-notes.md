# mysql-mcp v2.1.0

## Highlights
**mysql-mcp v2.1.0** delivers significant bug fixes, performance improvements, and enhanced testing coverage. This release focuses on production stability with fixes for Document Store, Group Replication, InnoDB Cluster, spatial tools, and MySQL Shell integration.

## Changed

- **Server Instructions** — Added document store filter syntax documentation with examples. Added spatial tools section documenting coordinate order behavior.
- **Partitioning Schema Descriptions** — Improved `value` parameter descriptions to clarify only boundary values should be provided.
- **Logger Tests** — Updated to match RFC 5424 severity levels (`warn` → `warning`).

## Removed

- **Jupyter Quickstart Notebook** — Removed `examples/notebooks/quickstart.ipynb` and the `examples/` directory due to kernel instability issues on Windows. Usage instructions are now provided via MCP protocol.

## Install

```bash
npm install -g @neverinfamous/mysql-mcp@2.1.0
docker pull writenotenow/mysql-mcp:v2.1.0
```
